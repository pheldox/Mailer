# Mailer
Web application to conduct surveys and take people's feedback.


## Project Details

Technologies/Tools:
* Reactjs for front-end
* Nodejs(Express) for backend/api
* MongoDB, mongoose. 
* Redux for state management
* Redux forms
* Materialize css
* Google Oauth
* Passport 
* Axios for making requests
* Lodash
* Stripe payment API
* SendGrid for emails
